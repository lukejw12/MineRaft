**Transforms your survival Minecraft experience into a recreation of the hit game 'Raft'**
This datapack comes with an overhauled experience, with new mechanics, new blocks, structures, and even new mobs!

**Since this is a Datapack and Resource pack, both can be installed once server side and will apply to all players that join, voiding the need for each player to install the pack!**

<details>
<summary>¹   How To Download!</summary>

Data Pack:
1. Click the latest Datapack version.
2. Scroll down and Download the Primary file.
3. If you want to create a World, go to Singleplayer > Create New World > Data Packs > Open Pack Folder.
(It is not recommended to put it in an already existing world!)
4. Copy/Cut and Paste the downloaded file there.
5. Open/Create your world.
6. Note: You need to also install the Required Resource Pack.
  
Required Resource Pack:
1. Click the Datapack featured version.
2. Scroll down and Download the Required Resource Pack.
3. Open Minecraft and go to Options > Resource Packs > Open Pack Folder.
4. Copy/Cut and Paste the downloaded file there.
5. Activate the Resource Pack.
(you can reload the resources with f3+t ingame)
